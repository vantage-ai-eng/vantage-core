# vantage-core 🚀

> **Unit tests pass on Turn 1. Vantage catches the drift by Turn 15.**

`vantage-core` is an open-source, lightweight developer SDK designed to simulate continuous, multi-turn interactions against your AI agents locally. While traditional evaluation frameworks test inputs and outputs in single-turn isolation, Vantage calculates performance decay curves over extended conversational states, catching system prompt regressions before code hits production.

## 📦 Installation

```bash
pip install .
```
